Title bar

features/editor/title-bar/ (D-007) — VS Code-style top chrome: app menu, command-center pill with the open document's name, layout toggles (the primary-sidebar toggle is functional; its band fills while a side panel is open), and window controls (close hovers #c42b1c, the Windows convention). Menus, command center, panel/secondary toggles, and window controls are static chrome in this first pass.

Document tree rows

features/documents/document-tree-node/ — shares the Explorer's tree-row style source. Document rows are draggable (drag onto a folder to move; the tree's empty space is the move-to-root zone); the row of the document open in the editor carries the active highlight and aria-current; a folder row lights up as a drop target mid-drag (shown frozen below). The raw ISO-8601 updatedAt lives in each document row's title tooltip.

Resize dividers
