Rules

Do Render timestamps, file names in tooltips, and identifiers exactly as stored — the app shows raw ISO-8601 strings, no date formatting.
Do Reserve uppercase + tracked type for the panel-header role (.tds-panel__header-title).
Don't Load webfonts. The system renders identically offline and over file://; platform faces are the contract.
Don't Use semibold for emphasis in running text — weight is a state signal (active document) and a hierarchy signal (titles), nothing else.

Single-line truncation

Names in rows and the command-center title never wrap: overflow: hidden; text-overflow: ellipsis; white-space: nowrap with the full value available via a native title tooltip. This is the one universal text-overflow policy.
