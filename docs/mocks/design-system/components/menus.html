Menus

One raised-list idiom covers everything that floats from a trigger: the title bar's app menus and the template picker's pop-out catalog. A raised surface, strong hairline, soft shadow, and quiet hover rows.

Menu surface

Derived from: templates/template-picker/ pop-out panel (the app's raised-list reference); adopted by the title-bar menus — .tds-menu.

Command center

Derived from: title-bar__command-center. Documented with the title bar in Toolbars & chrome; noted here because it is menu-adjacent: a pill-shaped trigger, not a menu itself.

Rules

Do Right-align supplementary text (shortcuts, categories) as the uppercase muted hint — one hint per item at most.
Do Return focus to the trigger when a menu dismisses.
Don't Nest submenus — Trellis menus are one level deep by design.
Don't Put icons in menu items; the raised list is a text idiom.
