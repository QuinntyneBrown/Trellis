Trellis — frontend HTML mocks

Static HTML mocks reverse-engineered from the Angular frontend (frontend/src/app). Every page mirrors the DOM the app actually renders — same component tags, class names, data-testids and ARIA attributes — styled by assets/mock.css, which is hand-compiled from the component SCSS. See README.md for the mock-to-component mapping and fidelity notes. Cards below are live 25% previews; click through for full size.
